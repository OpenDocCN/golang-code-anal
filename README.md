## 下载

### Docker

```
docker pull apachecn0/golang-code-anal
docker run -tid -p <port>:80 apachecn0/golang-code-anal
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g golang-code-anal
golang-code-anal <port>
# 访问 http://localhost:{port} 查看文档
```
